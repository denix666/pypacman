# pypacman
Clone of old DOS game "pacman".

The screenshot of the game:
![img](docs/screenshot1.png?raw=true "Screenshot")

## Requirements:
> Python >= v3.8

## How to run:
1. Install required modules

    > *pip install arcade*

2. Run the game:

    > *python main.py*
